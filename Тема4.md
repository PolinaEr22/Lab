## Самостоятельная работа №2

## Напишите программу, которая будет заменять игральную кость с 6 гранями. Если значение равно 5 или 6, то в консоль выводится «Вы победили», если значения 3 или 4, то вы рекурсивно должны вызвать эту же функцию, если значение 1 или 2, то в консоль выводится «Вы проиграли». При этом каждый вызов функции необходимо выводить в консоль значение “кубика”.

```python

import random

def roll_dice():
    dice = random.randint(1, 6)
    print(f"Выпало: {dice}")
    
    if dice >= 5:
        print("Вы выиграли!")
    elif dice >= 3:
        print("Продолжаем...")
        roll_dice()
    else:
        print("Вы проиграли.")

roll_dice()

```

## Выводы

Данный код представляет собой функцию roll_dice(), которая имитирует бросок игральной кости с 6 гранями. 

Каждый раз при вызове функции генерируется случайное число от 1 до 6 (с помощью random.randint(1, 6)) и выводится в консоль сообщение о выпавшем значении. 

Затем функция проверяет выпавшее число:
- Если выпало 5 или 6, выводится сообщение "Вы выиграли!"
- Если выпало 3 или 4, выводится сообщение "Продолжаем..." и функция вызывается рекурсивно для броска следующей кости.

Таким образом, данный код моделирует игру с броском кости, где игрок может выиграть, продолжить игру или проиграть в зависимости от выпавшего числа.

## Самостоятельная работа №3

## Напишите программу, которая будет выводить текущее время, с точностью до секунд на протяжении 5 секунд. Программу нужно написать с использованием цикла. Подсказка: необходимо использовать модуль datetime и time, а также вам необходимо как-то “усыплять” программу на 1 секунду.

```python
import time
import datetime

for _ in range(5):
    current_time = datetime.datetime.now()
    print(f"Текущее время: {current_time.strftime('%H:%M:%S')}")
    time.sleep(1)
```
### Результат.

![Меню](https://github.com/PolinaEr22/Lab/blob/Тема4/pic/3.png)


## Выводы

Данный код импортирует модули datetime и time. Использует for цикл для повторения блока кода 5 раз (for _ in range(5):). Переменная _ здесь используется как временная переменная цикла, которая не будет использоваться внутри цикла. Это стандартный способ показать, что переменная цикла не важна.
Внутри цикла, при каждой итерации, получает текущее время с помощью метода datetime.datetime.now(). Эта функция возвращает объект datetime, представляющий текущие дату и время.
Затем, используя метод strftime('%H:%M:%S'), преобразует объект datetime в строку, отформатированную в формате "ЧЧ:ММ:СС", где ЧЧ - часы, ММ - минуты, СС - секунды. 
Вызывает time.sleep(1), чтобы приостановить выполнение программы на 1 секунду перед тем, как перейти к следующей итерации цикла. Это означает, что между каждым отображением текущего времени будет промежуток в 1 секунду.

## Самостоятельная работа №4

## Напишите программу, которая считает среднее арифметическое от аргументов вызываемое функции, с условием того, что изначальное количество этих аргументов неизвестно. Программу необходимо реализовать используя одну функцию и “точку входа”.

```python
def average(*args):
    if len(args) == 0:
        print("Нет аргументов для вычисления среднего")
        return

    total = sum(args)
    avg = total / len(args)
    print(f"Среднее арифметическое: {avg}")

if __name__ == "__main__":
    nums = [float(num) for num in input("Введите числа через пробел: ").split()]
    average(*nums)
```

## Выводы
Данный код представляет собой простую программу на Python, которая вычисляет среднее арифметическое значение произвольного количества чисел, введенных пользователем. 
Вот как он работает пошагово:

1. Определяется функция average(*args), которая принимает произвольное количество аргументов (чисел), для которых будет рассчитываться среднее арифметическое. Звёздочка перед args означает, что функция может принять любое количество аргументов, которые будут доступны внутри функции в виде кортежа.

2. Внутри функции сначала проверяется, переданы ли в неё какие-либо аргументы (if len(args) == 0:). Если аргументы не переданы, выводится сообщение об отсутствии аргументов для вычисления и функция завершает свою работу.

3. Если аргументы переданы, сначала вычисляется их сумма с помощью встроенной функции sum(args), а затем эту сумму делят на количество аргументов (len(args)), чтобы получить среднее арифметическое.

4. После вычисления среднего арифметического значение выводится на экран.

5. В коде также присутствует проверка if name == "__main__":. Это условие проверяет, является ли данный файл программы главным исполняемым файлом. Если это так, то выполняется код внутри данного условия.

6. Программа предлагает пользователю ввести числа через пробел с помощью функции input(). Введенные значения разделяются методом split(), превращая строку в список строк.

7. Список строк преобразуется в список вещественных чисел nums с помощью list comprehension.

8. Наконец, функция average вызывается с распакованным списком чисел в качестве аргументов (*nums), чтобы вычислить и вывести среднее арифметическое введенных чисел.
